Exemplo de utilização

Neste exemplo utilizamos a tag 'header' junto com a tag 'nav' para criar um menu.
Utilizamos a tag 'main' para separar o conteúdo do site e dentro da tag 'main' utilizamos as tags 'section' e 'article' para separar os conteúdos.

'aside' é uma tag utilizada para referenciar ou relacionar algo ao artigo, como por exemplo o nome do autor do texto.

Essa seção é o rodapé do site. Exemplo: 'Site desenvolvido por...'
